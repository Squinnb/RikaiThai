# RikaiThai 🇹🇭

A rikaikun-inspired hover dictionary and word segmenter for Thai text — hover over any Thai word on the web and instantly see its definition, romanization, and part of speech.

## Features

- 🖱️ Hover over Thai text for instant definitions
- 🔍 Longest-match algorithm (rikaikun-style, cursor-first)
- 🎨 Three themes: Thai Flag, Midnight, High Contrast
- 🔛 Enable/disable toggle in the extension popup
- 🌐 Works on most sites including Google, Wikipedia, ChatGPT
- 📖 Dictionary sourced from kaikki.org (Wiktionary data)
- 🏷️ Includes 250 common Thai given and family names
